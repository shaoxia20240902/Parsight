Parsight / 析见

面向 XLSX 数据分析的智能 BI 应用。上传 Excel 后，AI 自动完成单表理解、跨表关联、看板生成与自然语言问答。

查看源码 快速开始

核心能力

从原始表格到可用洞察，Parsight 让数据分析不再依赖手动建模。

1

XLSX 上传与解析

支持 .xlsx / .xls，自动识别 Sheet 结构、字段类型并写入 MySQL。

2

AI 单表理解

六维理解业务含义，输出 Markdown 报告并支持异议核对与修正。

3

跨表关联分析

在空间维度发现多表关联，生成可核对的 SQL 与关系说明。

4

自动 BI 看板

基于理解结果自动推荐指标、维度与图表类型，生成可交互看板。

5

自然语言问答

快速洞察、深度研究、BI Builder 三种模式，用提问代替写 SQL。

6

空间与后台管理

多空间数据隔离、LLM 配置、用户与日志管理，适配团队使用。

三步获得洞察

无需数据仓库，无需预先建模。

1

上传 Excel

将 XLSX 文件拖到数据页，系统自动解析入库。

2

AI 自动理解

单表六维理解、跨表关联分析，生成业务语义与 SQL。

3

看板与问答

一键生成 BI 看板，或用自然语言继续追问数据。

快速开始

本地启动只需要 MySQL、Python 3.10+ 和 Node.js 18+。

git clone https://github.com/shaoxia20240902/Parsight.git
cd Parsight/backend && cp .env.example .env
# 编辑 .env：配置 DATABASE_URL 与 JWT_SECRET_KEY
cd .. && ./start.sh

启动后访问 http://localhost:3000，默认管理员账号 admin，首次启动密码见后端日志。LLM 配置请在 /admin/llm-config 完成。
