# Open Source Grant Notes

Parsight is a solo-maintained MIT open-source project built to make AI BI workflows accessible to teams that still manage business data in spreadsheets. It combines XLSX ingestion, semantic sheet understanding, cross-table relationship discovery, SQL generation, ECharts dashboards, natural-language Q&A, and report generation in one inspectable full-stack application.

The project is a strong fit for developer support programs because it is not only a demo: it includes backend services, frontend UI, database integration, agent orchestration, streaming progress, mock data, tests, architecture documents, and contribution guidance. It is designed for continuous improvement and public collaboration.

Additional support would directly accelerate work on security review, SQL safety, model evaluation, dashboard quality, and API cost control. The project can also serve as a practical reference for other developers building AI applications that connect LLM reasoning with private structured data.
