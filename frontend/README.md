# Parsight Frontend

Vue 3 + Vite 前端，提供 XLSX 管理、数据理解、BI 看板与聊天洞察界面。

## 环境

- Node.js 18+

## 启动

```bash
cd frontend
npm install
npm run dev
```

开发服务器默认运行在 `http://localhost:3000`，代理后端 `/api` 到 `http://localhost:8007`。

## 构建

```bash
npm run build
npm run test
```

完整项目说明见 [../README.md](../README.md)。
