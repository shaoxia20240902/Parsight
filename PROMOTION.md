## Hacker News — Show HN

**标题**：Show HN: Parsight – Turn Excel files into an AI-powered BI dashboard

**正文**：

```text
Parsight is an open-source BI app for XLSX files. Upload one or more Excel sheets and it automatically:

- Parses structure and loads data into MySQL
- Uses an LLM agent to understand each table's business meaning
- Discovers cross-table relationships and generates SQL
- Recommends metrics/dimensions and builds interactive dashboards
- Answers natural-language questions in three modes: quick insight, deep research, BI builder

Stack: FastAPI, SQLAlchemy, Vue 3, Vite, Element Plus, ECharts. Supports DeepSeek / DashScope LLMs.

Demo / screenshots: https://shaoxia20240902.github.io/Parsight/
Repo: https://github.com/shaoxia20240902/Parsight
```
